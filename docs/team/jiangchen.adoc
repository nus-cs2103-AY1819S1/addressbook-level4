== PROJECT: ExpenseTracker

== Overview

This portfolio aims to document my role and contribution to the *ExpenseTracker*.

This project is CS2103T team project. The main purpose of this project is to morph
AddressBook - Level 4,a given desktop application, into a new application.
ExpenseTracker is a desktop application used for tracking user's expenses.
The user interacts with it using a CLI, and it has a GUI created with JavaFX.
It is written in Java, and has about 20 kLoC.

My contribution to the project included

== Summary of contributions

* *Major enhancement*: added *the ability to find and edit multiple expenses by keywords*
** What it does: Allow user to filter out the particular expenses by entering relevant keywords. Allow user to mass edit
these expenses.
** Justification: This feature improves the product significantly because it is more convenient for user to find and edit particular expenses.
** Highlights: This enhancement affects existing commands and commands to be added in future.
It required an in-depth analysis of design alternatives. The implementation too was challenging
as it required changes to existing commands.
