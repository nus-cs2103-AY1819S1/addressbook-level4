== Overview
*A desktop Event Scheduler application.*


* *CLI (Command Line Interface)* interactions with modern *GUI* brought to you by us!
* *Always connected.* Sync it with your Google account!
* *Cross platform.* You can run it on both *Windows* AND *Mac*!
* *Responsive interface.* We ensure your smooth experience.

== Latest Features
* Supports repeating events.
* Supports tag management.
* Syncs with Google services.
* Pop-up alert.
