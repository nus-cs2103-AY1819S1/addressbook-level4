* This is a desktop version of LearnV. It has a GUI but most of the user interactions happen using a CLI (Command Line Interface).
* It is a Java sample application intended for students learning Software Engineering while using Java as the main programming language.
* Compared to similar existing vocabulary learning application, this application is specially designed for those who have the habit of using CLI to learn new vocabulary.
